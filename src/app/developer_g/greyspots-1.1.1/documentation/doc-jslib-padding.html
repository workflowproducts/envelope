String Padding

Functions:

GS.rightPad(<STRING-TO-PAD>, <STRING-TO-PAD-WITH>, <LENGTH-TO-PAD-TO>);
GS.leftPad(<STRING-TO-PAD>, <STRING-TO-PAD-WITH>, <LENGTH-TO-PAD-TO>);

Description:

The GS.rightPad and GS.leftPad functions are for adding characters to a string to make it a certain length or longer. The primary use of this is for when creating a text representation of a date object. You may want a date that looks like this: "1-1-2015" to look like this: "01-01-2015" or you may want a date that looks like this: "12-1-2015" to look like this: "12-01-2015". This is created using padding functions.
