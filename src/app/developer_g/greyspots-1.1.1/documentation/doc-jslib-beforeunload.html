Add Before Unload Event

Function:

GS.addBeforeUnloadEvent(function () {...});

Description:

Javascript has a feature which allows the developer to run code before the page unloads, this is used primarily as a means of opening up a popup asking to confirm if you want to close the page. This feature is accessed using the window.onbeforeunload property. However there is a limitation of this property: you cannot run two independent functions using the same property.

This function was created to allow multiple "onbeforeunload" functions to run. Just call the function and send your function that you want to bind and it'll add it to the stack.
