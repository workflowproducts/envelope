Function:

GS.cloneElement(<ELEMENT>[, <DOCUMENT>]);

Description:

The GS.cloneElement function is used for cloning an element. This function also works around an issue on iOS where template elements have cloning issues.

Examples:

GS.cloneElement Example:

In this example when you click a "Clone" button a copy of the button you clicked is appended to the end of the document.

GS.cloneElement Custom Document Example:

When you click the "Custom Document" button a copy of the button will be created in the iframe element's document. There is no visual component to this example. This example is only so that you can see how to use it and what it's for.
