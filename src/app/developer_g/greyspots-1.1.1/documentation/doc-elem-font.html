Font

Tag:

<gs-font></gs-font>

Description:

The font tag allows you to change the font size of text based on window size or media query.

Examples:

Skeleton Example:

A basic gs-font element. This font element will have the following behaviors:

  1. When the window is above 0 width the font-size will be set to 10px
  2. When the window is above or equal to 500px width the font-size will be set to 20px

Attribute [min-width]:

This "min-width" attribute enables you to change the font size depending the minimum width of the screen. The "min-width" attribute takes a list of expressions seperated by semi-colons. The format of an expression is "<WINDOW-WIDTH> {<FONT-SIZE>};". Precedence for the font-sizes is determined by order, so the last expression is more important that the first.

There are several shortcuts for some pre-set sizes:
  1. All instances of "all" are replaced with "0px"
  2. All instances of "small" or "sml" are replaced with "768px"
  3. All instances of "medium" or "med" are replaced with "992px"
  4. All instances of "large" or "lrg" are replaced with "1200px"
In this example the gs-font element will change the font-size at 4 widths.
Attribute [media]:

The "media" attribute enables you to change the font size depending CSS Media Queries. The "media" attribute takes a list of expressions seperated by semi-colons. The format of an expression is "<MEDIA-SELECTOR> {<FONT-SIZE>};". Precedence for the font-sizes is determined by order, so the last expression is more important that the first.

There are several shortcuts for some pre-set sizes:
  1. All instances of "small" or "sml" are replaced with "768px"
  2. All instances of "medium" or "med" are replaced with "992px"
  3. All instances of "large" or "lrg" are replaced with "1200px"
In this example the gs-font element will change the font-size at 4 media conditions.
