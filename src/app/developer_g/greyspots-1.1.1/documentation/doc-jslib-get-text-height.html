Text Height Calculation

Function:

GS.getTextHeight([<SCOPE>[, <USE-NORMAL-LINE-HEIGHT>]]);

Description:

The GS.getTextHeight function is for getting the height of text either in the body or in an element of your choice.

Examples:

Skeleton Example:

In this example we are getting the text height of the body element and putting it into the "result" div so that you can see.

Specific Element Example:

In this example we are getting the text height of specific elements instead of the body. Some of these different elements have their font-size set which changes the text

height.
CSS line-height Example:

In this example we have an element with a custom CSS line-height set that we want to get the line height of. In order for the GS.getTextHeight function to do this we have to tell it to respect line-height by sending true as the second parameter. If you remove the second parameter we get the height of the text instead of the height if the line.
