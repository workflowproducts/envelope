Find Parent

Functions:

GS.findParentTag(<ELEMENT-TO-SEARCH-FROM>, <TAG-NAME>);
GS.findParentElement(<ELEMENT-TO-SEARCH-FROM>, <SELECTOR|FUNCTION|ELEMENT>);

Description:

These functions are for finding a parent element of a specified element.

Examples:

GS.findParentTag Example:

This function finds a parent using the tag name. It loops through parents and stops when it finds the first element with the specified tag name. In this example you choose a tag to find using the select box and when you click the "Find Parent" button it will find the parent with that tag and put a CSS border on it.

GS.findParentElement Selector Example:

In this example we use the GS.findParentElement function to find a parent using a selector. It loops through parents and stops when it finds the first element that the selector matches. In this example you choose an ID to find using the select box and when you click the "Find Parent" button it will find the parent with that ID and put a CSS border on it.

GS.findParentElement Function Example:

In this example we use the GS.findParentElement function to find a parent using a provided function. It loops through parents and stops when the provided function returnes true. In this example when you click the "Find Parent" button the GS.findParentElement function will find the parent with that the function returns true on and put a CSS border on it.

GS.findParentElement Element Example:

In this example we aren't searching for a parent because we already have that element. We are searching to see if the target is a child of that element.
