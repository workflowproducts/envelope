Container And Jumbo

Tags:

<gs-container>...</gs-container>

<gs-jumbo>...</gs-jumbo>

<gs-jumbo>
    <gs-container>...</gs-container>
</gs-jumbo>

Description:

These elements provide desktop and mobile aware containers for your applications. These containers are useful for building static webpages that look good on phones, tablets and desktop devices.

Examples:

Skeleton Container Example:

A basic <gs-container> element with a header and paragraph. It will resize to the window and flow text.

Attribute "padded":

To add padding to a <gs-container> you just add the "padded" attribute.

Attribute "min-width":

The "min-width" attribute is for telling the <gs-container> to snap to particular width when the window reaches certain points. In this example the container will change it's width when the window is 300px or wider, 500px or wider and 700px or wider.
 Note 1: resize the window to see this behaviour.
 Note 2: When the window snaps to a width (e.g. 500px) it might (depending on OS and OS settings) be a little thinner than 500px to make room in case there is a scrollbar next to the container.

The "min-width" attibute takes a list of widths seperated by semi-colons. Precedence for the font-sizes is determined by order, so the last width is more important that the first.

Here are some shortcuts for pre-set widths:
  1. All instances of "small" or "sml" are replaced with "768px"
  2. All instances of "medium" or "med" are replaced with "992px"
  3. All instances of "large" or "lrg" are replaced with "1200px"
Attribute "media":

The "media" attribute is for telling the <gs-container> to snap to particular width based on CSS Media Queries.

The "media" attribute takes a list of expressions seperated by semi-colons. The format of these expressions is "<MEDIA-SELECTOR> {<WIDTH>};". Precedence for these expressions is determined by order, so the last expression is more important that the first.

Here are some shortcuts for pre-set widths:
  1. All instances of "small" or "sml" are replaced with "768px"
  2. All instances of "medium" or "med" are replaced with "992px"
  3. All instances of "large" or "lrg" are replaced with "1200px"
