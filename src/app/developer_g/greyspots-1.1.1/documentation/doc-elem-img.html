Image

Tag:

<gs-img src="..."></gs-img>

Description:

The gs-img element is for adding an image to your page and having it change it's size depending on window width or CSS Media Queries.

Examples:

Skeleton Example:

A basic gs-img element. This element will have the following behaviors:

  1. The gs-img will center itself
  2. The gs-img will always be 272px wide by 478px tall
Attribute [min-width]:

The "min-width" attribute enables you to change the images size depending the minimum width of the screen. The "min-width" attribute takes a list of expressions seperated by semi-colons. The format of an expression is "<WINDOW-WIDTH> {<IMAGE-WIDTH>, <IMAGE-HEIGHT>};". If no image height is specified: the gs-img will use the image width as the image height. Precedence for the expressions is determined by order, so the last expression is more important that the first.

Here is a list of shortcuts for some pre-set sizes:
  1. All instances of "all" are replaced with "0px"
  2. All instances of "small" or "sml" are replaced with "768px"
  3. All instances of "medium" or "med" are replaced with "992px"
  4. All instances of "large" or "lrg" are replaced with "1200px"
In this example the gs-font element will change the image size at 4 widths.
Attribute [media]:

The "media" attribute enables you to change the image size depending CSS Media Queries. The "media" attribute takes a list of expressions seperated by semi-colons. The format of an expression is "<MEDIA-SELECTOR> {<IMAGE-WIDTH>, <IMAGE-HEIGHT>};". Precedence for the expressions is determined by order, so the last expression is more important that the first.

There are several shortcuts for some pre-set sizes:
  1. All instances of "small" or [sml] are replaced with "768px"
  2. All instances of "medium" or [med] are replaced with "992px"
  3. All instances of "large" or [lrg] are replaced with "1200px"
In this example the gs-font element will change the font-size at 4 media conditions.
Attribute [image-cover]:

The "image-cover" attribute enables you to fill the gs-img even if some of the image will get chopped off.

Attribute [align]:

The "align" attribute enables you to make the image go to the left or right instead of center.
