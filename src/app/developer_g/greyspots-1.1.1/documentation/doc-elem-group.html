Group

Tag:

<gs-group name="..."></gs-group>

Description:

The gs-group element is used to contain and label a set of related elements.

Examples:

Skeleton Example:

A basic gs-group element. The "name" attribute is required and it determines the text for the label below the box.

[padded] Attribute:

To put some space around a gs-group element you use the "padded" attribute.
