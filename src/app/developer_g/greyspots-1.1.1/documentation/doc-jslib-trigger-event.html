Trigger Event Function

Function:

GS.triggerEvent(<ELEMENT>, <EVENT-NAME>, <CONFIGURATION>)

Description:

This function used is for triggering events via javascript. The events can either be standard events (Like: "click", "keydown", "mouseover", "load" etc...) or custom events. Custom events are just events with your own custom name.

Examples:

Skeleton Example:

In this example we will have a button that will either recieve a click from the mouse or recieve a click from javascript.

Configuration Example:

In this example we will set properties on a custom event. These properties are accessed the same way as the standard EVENT.target property. The use of this is to pass data to event listeners using an already established standard: using the EVENT object.
