Scroller

Tag:

<gs-scroller>
    <gs-scroller-inner style="width {WIDTH}; height: {HEIGHT};">{CONTENT}</gs-scroller-inner>
</gs-scroller>

Description:

The gs-scroller element is for navigating around content by clicking and dragging to move and using the scroll wheel to zoom in and out.

Notes:
  • On a phone this element is only scrollable, not zoomable.
  • The gs-scroller element must have it's contents inside a gs-scroller-inner element.
  • The gs-scroller element changes it's zoom by adjusting font-size, so make sure your content responds to font-size (including any width/height on the gs-scroller-inner).
