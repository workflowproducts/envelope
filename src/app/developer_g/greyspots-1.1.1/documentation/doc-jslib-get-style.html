Get Style

Function:

GS.getStyle(<ELEMENT>, <CSS-PROPERTY>);

Description:

This function is for getting a "computed" CSS property of an element. The "computed" style is the style that the browser decides on after reading the CSS. So if "background-color" is set three times on the same element and the browser chooses to use the third which we'll just say is "#0F0" (solid green) that would be the computed style. Notice: the browser may translate the choosen value to another format so it may turn "#0F0" into "rgb(0, 255, 0)" and that'll show through as the computed style.

Examples:
GS.getStyle Example:

In this example when you click the "Get Target 1 Style" button the "result" div will be populated with the computed "background-color" style of the "Target 1" element. When you click the "Get Target 2 Style" button the "result" div will be populated with the computed "background-color" style of the "Target 2" element.
