Element Position

Functions:

GS.getElementOffset(<ELEMENT>);
GS.getElementPositionData(<ELEMENT>);

Description:

These functions are for getting data about the position of an HTML element.

Examples:

GS.getElementOffset Example:

This function is for getting the left and top of an element relative to the viewport. This function is useful for example for positioning a dropdown box near an element.

GS.getElementPositionData Example:

This function is for getting:
    Element Width
    Element Height
    Element Top (relative to the viewport)
    Element Left (relative to the viewport)
    Element Bottom (relative to the viewport)
    Element Right (relative to the viewport)
    Room Above (relative to the viewport)
    Room Below (relative to the viewport)
    Room To The Left (relative to the viewport)
    Room To The Right (relative to the viewport)
