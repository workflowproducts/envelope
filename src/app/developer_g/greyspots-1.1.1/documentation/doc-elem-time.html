Time

WARNING: This element is deprecated, please use the <gs-datetime></gs-datetime> instead

Tag:

<gs-time></gs-time>

Description:

The gs-time control is like the gs-text control, but it is specialized for times.

Examples:

Skeleton Example:

A basic gs-time element. Without attributes the gs-time control will have the following features/behaviors:

  1. ) There is a button that activates the time picker.

  2. ) If the time picker is in modal mode: the time picker can be dismissed by clicking the "Done" button. If the time picker is not in modal mode: clicking out of the time control will dissmiss the time picker.

  3. ) You can also change the time manually.

Attribute [column=""]:

When you have a gs-time inside a data element and you want to use it to display cell data you use the column attribute.

Attribute [value=""]:

To set the value for the gs-time, use the "value" attribute.

Attribute [qs=""]:

The name [qs] stands for "querystring". The querystring is the part in the URL after the "?". The [qs] attribute has two formats, each with their own use.

  • qs="key1=attribute1". This format allows you to fill any attribute with any key from the querystring. To use multiple keys from the querystring you can put a comma-separated list of these expressions (for example: qs="key1=attribute1,key2=attribute2"). If you need an attribute to be added when a querystring key is not present, replace the "=" with "!=" (for example: qs="key1!=attribute1", attribute1 will only be added if key1 is not in the querystring). As a shorthand, if the key in the querystring is the same name as the attribute you want to fill, you can omit the attribute name in the expression (for example: qs="key1="). If an expression doesn't find it's key in the querystring: it reverts the attribute that it fills to it's original value.
  • qs="key". This format will update the [value] attribute with the querystring whenever the querystring changes.
Attribute [Placeholder=""]:

The "placeholder" attribute specifies some placeholder text for when the control is empty.

Attribute [mini]:

Sometimes you are limited on space. The mini attribute is used to make the gs-time smaller.

Attribute [no-picker]:

If you want to remove the date picker button: add the "no-picker" attribute.

Attribute [no-now-button]:

If you want to remove the "Now" button from the picker: add the "no-now-button" attribute.

Attribute [non-empty]:

If you want to force the user to input a value: add the non-empty attribute. Note: if a default value is not provided then the control defaults to "12:00 PM".

Attribute [format=""]:

The "format" attribute determines the display format of the control. This attribute can take two different values: "regular" and "military". "regular" is normal 12-hour time and "military" is 24-hour time.

Attribute [output-format=""]:

The "output-format" attribute determines the internal format of the control. This attribute can take two different values: "regular" and "military". "regular" is normal 12-hour time and "military" is 24-hour time. When getting the value of the control in javascript "output-format" is the format it will get.

Attribute [tabindex=""]:

With the "tabindex" attribute, you can control what order your controls are tabbed to.

Attribute [readonly]:

The [readonly] attribute is similar to [disabled] because it prevents the value from changing. The differences are that readonly is styled differently and allows you to copy the current text from the control.

Javascript Methods:

These functions can be run on individual <gs-time> elements:
TIME.open() <-- This method opens the <gs-time>.
