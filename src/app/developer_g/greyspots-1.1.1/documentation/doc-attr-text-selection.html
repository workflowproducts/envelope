Text Selection Attributes

Attributes:

prevent-text-selection
allow-text-selection

Description:

These attributes are for preventing and enabling text selection on elements.

Examples:

Attribute "prevent-text-selection":

In this example we have two divs: one that can have it's text selected and one that cannot. To prevent the text selection on the second div the "prevent-text-selection" attribute was put on it.

Attribute "allow-text-selection":

In this example we have two divs, one inside the other. The outer div has text selection prevented by using the "prevent-text-selection" attribute, but the inner div has it's re-enabled by the "allow-text-selection" attribute.
