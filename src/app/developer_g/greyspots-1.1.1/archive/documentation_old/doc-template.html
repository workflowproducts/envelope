doT.js

Description:

Many of our webcomponents use doT.js for record templates (among other things). For general information about doT.js go to http://olado.github.io/doT/

< and >


Sometimes you have a doT.js if like so:

{{? i < len }}
    // do stuff
{{?}}

But when the javscript gets to it, it looks like this:

{{? i &lt; len }}
    // do stuff
{{?}}

So we wrote the following functions:

gt(x,y):  x >  y
gte(x,y): x >= y
lt(x,y):  x <  y
lte(x,y): x <= y

Record templates


To address a column use row.<column>:

{{! row.test1 }}
 OR 
{{= row.test1 }}
