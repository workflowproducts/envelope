Tooltip

Tags:

<gs-tooltip></gs-tooltip>

Description:

The gs-tooltip is used to display a tooltip instantly when its target is hovered over or clicked. The target is set using the for attribute, or by wrapping the target in the gs-tooltip element. If no target is found the gs-tooltip will automatically set the target to the element directly after itself in the DOM at the time of its initialization.

The message displayed can be set using the tip attribute or in the case of using the the for attribute rather than wrapping the target, you can place the message text inside the gs-tooltip.

Examples:

Skeleton Example:

A basic gs-tooltip element. The tooltip has the following behavior:

  1. ) When the target of the gs-tooltip fires the mouseover or click event the gs-tooltip will become visible next to the target.

  2. ) When the target of the gs-tooltip fires the mouseout or the document.body fires the click event the gs-tooltip will become invisible.

Attribute "tip":

The onclick attribute allows you to run Javascript as a result of clicking on the tooltip.


Directions:

The default direction of the gs-tooltip message can be set using the left, right, top, and bottom attributes. The default direction is right. If there is not room for the message the message will be displayed in a different direction
