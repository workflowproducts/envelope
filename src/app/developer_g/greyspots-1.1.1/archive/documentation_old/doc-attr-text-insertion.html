Text Insertion Attributes For Mobile Devices

Attributes:

autocorrect
autocapitalize
autocomplete
spellcheck

Description:

These attributes are for preventing text insertion behaviours for elements on mobile devices.

Examples:

Attribute [autocorrect=""]:

This atribute is for removing mobile devices default autocorrect behaviour.

Attribute [autocapitalize=""]:

This atribute is for removing mobile devices default autocapitalize behaviour.

Attribute [autocomplete=""]:

This atribute is for removing mobile devices default autocomplete behaviour.

Attribute [spellcheck=""]:

This atribute is for removing mobile devices default spellcheck behaviour.
