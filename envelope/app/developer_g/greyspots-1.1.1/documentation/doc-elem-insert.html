Insert Record

Tag:

<gs-insert src="...">...</gs-insert>

Description:

This element is for inserting records in the database.

Examples:

Skeleton Example:

This is a basic <gs-insert> element. When the "submit" method is called the element makes an ajax call for the table or view named in the "src" attribute and attempts to insert to it.

Attribute [addin]:

The "addin" attribute adds data to the insert that the user does not see. In this example the "first_name" column will be inserted with the value "Blackberries" even though the user did not type it in.

Attribute [addin] And The Query String:

The "addin" attribute can connect to the query string using doT.js. In this example the "first_name" column will get it's value from the query string column "people_id"

Attribute [pk]:

The [pk] attribute defines the PRIMARY KEY column of the object.

Attribute [seq]:

The [seq] attribute contains the sequence used for the PRIMARY KEY specified in the [pk] attribute.
